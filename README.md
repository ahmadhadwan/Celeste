Celeste
=======

# Dependencies

## Linux

- cglm development files (C headers)

- glfw

- openal

## Windows

Dependencies are already available in the winbuild directory.

- cglm development files (C headers)

- OpenAL32.dll

# Building

Building requires a C compiler, ar, and make.

## Linux

```
    make
```

## Windows
```
    make win
```
